# cfg 的格式

双斜杠是添加评论 以下内容都以评论的格式来写。  

第一行 写使用英雄的中文名称

第二行写 需要解除绑定的按键

第三行 描述一下每个按键的功能。

最后一行留下这篇cfg 作者名称， 也可以不写。

之后添加 cfg 的正文内容
